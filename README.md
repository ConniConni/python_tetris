☆pythno の実行環境があれば遊べます。

1. 動作確認済みバージョン
   python 3.11.7

2. 実行手順
   ① ソースを一式ディレクトリに配置
   ②python main.py を実行
   ③ ゲーム画面が立ち上がったら start ボタンを押してゲームスタート

3. 操作説明
   キーボードで操作します。対応するアクションとキーは以下の通りです。
   移動　　　 →、↓、←
   回転　　　 ↑
   ホールド　シフト(右)
   入れ替え　シフト(右)

4. 得点
   １列消し：100 点
   ２列消し：300 点
   ３列消し：500 点
   ４列消し：800 点
